# KongsiLogi Logo

Silakan masukkan file logo Anda ke dalam folder ini.
Direkomendasikan menggunakan format `.svg`, `.png`, atau `.jpg` transparan dengan nama:
- `logo.svg` atau `logo.png`

Nantinya logo ini dapat dipanggil di komponen React seperti:
```tsx
<Image src="/assets/logo/logo.png" alt="KongsiLogi Logo" width={150} height={40} />
```
